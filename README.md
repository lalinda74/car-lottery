# CarLottery
Technologies and tools: 
- [ ] Angular 13
- [ ] SCSS
- [ ] TypeScript
- [ ] NgRx
- [ ] Angular Material
- [ ] Bootstrap 5

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

To run the mock back-end served, run `npm run mock:server` in a console.
